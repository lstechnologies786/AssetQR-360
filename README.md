# 📖 Overview

AssetQR 360 is an enterprise-grade QR-Based Asset Management, Inspection, and Tracking Platform developed by **LS Technologies**.

The platform enables organizations to digitally manage their assets through secure QR codes, smart inspections, GPS verification, cloud synchronization, and real-time analytics.

AssetQR 360 helps educational institutions, enterprises, industries, healthcare organizations, government departments, and corporate offices efficiently monitor the complete lifecycle of their assets.

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/AssetQR-360.git
```

Go inside project

```bash
cd AssetQR-360
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run

```bash
python app.py
```
